# User Roles

## Visitor

Browse publicly available content.

## User

Create profile, participate in community, message vendors.

## Vendor

Sell products/services, receive payments, manage listings.

## Directory Listing

Public informational listing only.

## Administrator

Moderate content, manage users, verify vendors.

**Account hubs by role**

| Role | Hubs shown |
|------|------------|
| Member | Vitals, Member Hub |
| Approved vendor | Vitals, Member Hub, Vendor Hub, GrowSpace |
| Admin | All five (adds Admin Hub) |

Admin Hub (`/account?hub=admin-hub` → `/account/admin/*`) is only visible to users with the `ADMIN` role.
