# RootSync Project Overview

## What is RootSync?

RootSync is a platform that helps people discover, buy, sell, learn, and
connect around locally produced goods and services.

## Current Pillars

-   Marketplace
-   Community
-   AI Assistant
-   Messaging
-   Interactive Map
-   Digital Products
-   Consultations

## Success Metric

Help local businesses and communities thrive by making local commerce
easier, more connected, and more discoverable.
